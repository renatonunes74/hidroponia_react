## Como usar
1. Clone o repositório
    - `git clone https://github.com/renatonunes74/hidroponia_react`
2. Entre na pasta
    - `cd hidroponia_react`
3. Instale os módulos
	- `npm i`
4. Rode localmente
    - CLI / via Terminal:
        - `npm start`
5. Agora é só acessar via navegador o endereço: `localhost:3000`
